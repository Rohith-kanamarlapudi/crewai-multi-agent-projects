# CrewAI Multi-Agent Projects

A collection of practical **CrewAI** projects demonstrating real-world multi-agent collaboration using autonomous AI agents.

## Projects

| Project | Description |
|---------|-------------|
| 💻 Coder | AI software engineering and code generation workflow |
| 🗣️ Debate | Multi-agent debate and reasoning system |
| 🏗️ Engineering Team | Collaborative engineering workflow using specialized agents |
| 📈 Financial Researcher | AI-powered financial analysis and research |
| 📊 Stock Picker | Intelligent stock screening and recommendation system |

## Technologies

- CrewAI
- Python
- YAML
- LLM APIs
- Custom Tools
- Knowledge Bases

Each project follows a modular architecture with configurable agents, reusable tools, and structured workflows, making it easy to extend and adapt to new use cases.
